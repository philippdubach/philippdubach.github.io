In this tutorial I'm going to walk through how to install [YOURLS](https://yourls.org) a powerful set of PHP scripts that allows you to run Your Own URL Shortener on your server.

<details>
    <summary>Prerequisites</summary>
    - Custom Domain <br>
    - Acsess to Domain DNS<br>
    - VM Instance (In this case with Ubuntu 22.04)<br>
    - SSl Certificate (preferably)<br>
    - Latest [YOURLS release archive](https://github.com/YOURLS/YOURLS/releases)
</details>
<br>

you can find what suits you on the [Cloud Free Tier Comparison List](https://github.com/cloudcommunity/Cloud-Free-Tier-Comparison?tab=readme-ov-file#1-aws). I Opted for Azure since I already had experience with AWS and Google Cloud
